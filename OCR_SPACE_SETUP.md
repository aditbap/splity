# OCR.space API Setup

To use the free OCR.space API, you need an API Key.

1. Go to [OCR.space Free API Key](https://ocr.space/ocrapi/freekey).
2. Enter your email address.
3. Check your email for the API Key (usually arrives instantly).
4. Add the key to your `.env.local` file:

```env
OCR_SPACE_API_KEY="helloworld" 
# Use 'helloworld' for testing/demo, or your real key for production
```

**Note**: The free tier has a limit of 25,000 requests per month.
